CARBURIO - VERSION V8 CORRIGEE

Correction importante :
- l'API officielle utilise e10_prix, sp95_prix, gazole_prix...
- l'ancien code cherchait surtout prix_e10, prix_sp95...
- maintenant les deux formats sont acceptés.

Fonctions :
- prix carburants
- noms / enseignes des stations quand disponibles
- distance
- carte
- curseur distance de 5 km à 80 km
- logo Carburio
